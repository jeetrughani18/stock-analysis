# Stock Performance Analysis

A Streamlit web app that analyses stock performance against benchmark indices using historical data from Yahoo Finance.

## Features
- Rolling Beta, Information Ratio, Sortino Ratio, Calmar Ratio
- Maximum & Average Drawdown (1Y / 3Y / 5Y)
- Days to Recovery
- Return Spread & Outperformance Days
- Normalised price chart (last 5 years)
- CSV export

## Run Locally
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```
